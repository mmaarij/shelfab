## 📖 Overview

**Shelfab** is a modern, local-first EPUB library manager. Built for readers who want to sync their **StoryGraph** library seamlessly, manage e-book metadata effortlessly, and bring the beauty of a physical bookshelf to their digital collection. 

Unlike other robust but clunky tools, Shelfab is designed to feel native and elegant while remaining strictly **Open Source (GPLv3)** and **Privacy First**. Your StoryGraph username and library data never leave your machine except when directly interacting with metadata providers. No analytics, no tracking.

## ✨ Features

### Elegant Metadata Management
Edit titles, authors, series info, and descriptions perfectly. Shelfab automatically updates and refactors your EPUB files internally so your files carry the accurate data anywhere.

<img src="https://shelfab.app/sc-edit-details.png" width="800" />

### Beautiful Cover Arts
Say goodbye to generic covers. Search the web for high-resolution artwork directly within the app, or upload your own custom covers to make every book stand out.

<img src="https://shelfab.app/sc-cover-lookup.png" width="800" />

### Automatic Series Grouping
Keep your collections together. Shelfab automatically pulls series information and groups related books in a beautiful, animated timeline so you always know what to read next.

<img src="https://shelfab.app/sc-series.gif" width="800" />

### Seamless StoryGraph Integration
Sync your reading progress and managed library effortlessly. Take full control of your local collection with intuitive settings and a single, organized directory.

<img src="https://shelfab.app/sc-settings.png" width="800" />

## 🚀 Installation

Download the latest release for your platform from the [Releases page](https://github.com/mmaarij/shelfab/releases).

- **Windows:** Download the `.exe` installer. *(If SmartScreen blocks it, click 'More Info' -> 'Run Anyway')*
- **macOS:** Download the `.dmg`. *(If Gatekeeper blocks it, right-click and select 'Open')*
- **Linux:** Download the `.deb` (Debian/Ubuntu) or `.rpm` (Fedora/RHEL) packages.

*Note: Shelfab is developed by an independent developer and is currently unsigned to keep the project completely free. This is why you might see standard OS warnings on your first install.*

## 🛠️ Development

If you want to build Shelfab from source, you'll need [Node.js](https://nodejs.org/) installed:

```bash
# Clone the repository
git clone https://github.com/mmaarij/shelfab.git
cd shelfab

# Install dependencies
npm install

# Start the app in development mode
npm run start

# Package the app for your OS
npm run make
```
